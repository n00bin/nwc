## Week of 2026-05-04

### Bug Fixes
- **Insignia Sharing Plan now allows cross-loadout mount reuse.** Previously, if a mount was picked for one bonus (e.g., Crimson Crystal Horse for Cautious Devotion in a Healer loadout), the planner refused to suggest it for a different bonus in another loadout (e.g., Guardian's Spirit in a Tank loadout) — even though only one loadout is active at a time. Tracking is now per-loadout: a mount only "conflicts" if it's already picked for a different bonus *in the same loadout*.
