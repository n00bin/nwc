## Week of 2026-05-29

### Bug Fixes
- Mounts: 4-slot mounts now show every compatible insignia bonus. The matcher was ignoring the 4th insignia slot for bonuses that need 3 or fewer insignias (a leftover from an old assumption that slot 4 was "stats-only"). All 4 slots count toward bonuses in-game, so 4-slot mounts like Snowtusk were hiding valid bonuses — e.g. Snowtusk now correctly shows Assassin's Covenant and Champion's Return alongside Gladiator's Guile. Affects all 51 four-slot mounts.
